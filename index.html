UsabLog is an experimental usability test observation logging tool built by Will Sansbury. I'm building UsaBlog partly to fill a need I've felt for easy, lightweight usability test logging and partly as a learning exercise for me to improve my coding skills.

Use UsabLog at your own risk. I make no guarantees or warranties about this tool.

UsabLog using HTML5 local storage to store logs. That means if you clear your browser history, you will lose your logs. I strongly recommend you copy and paste the log into another format after you log a session.

Interested in contributing to UsabLog's development effort? Check out UsabLog on GitHub.

In addition to logging notes, you can use these commands to do a few other things.

/t {Task Name}
Log the beginning of a new task.
/erase
Remove the last log entry. Be warned, though: you erase it, it's gone, and there isn't any confirmation.
/nuke
Blow away all localStorage data. This is useful for returning to a complete state. If you do this, you will lose ALL sessions you've logged, and you won't be able to recover them.
